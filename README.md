# Easy-Coding 任务作业：全屏贪吃蛇游戏系统

这是一个基于 **Vibe-Coding** 理念开发的 Web 项目，作为 **Easy-Coding** 的任务作业完成。本项目实现了一个具有用户系统、在线排行榜和高度可定制化的全屏贪吃蛇游戏。

## 🚀 项目功能

- **全屏游戏体验**：适配各种屏幕尺寸的沉浸式贪吃蛇游玩。
- **用户认证系统**：
  - 支持注册与登录。
  - **唯一性设计**：注册时自动分配 `昵称#UUID后四位`，确保全球唯一标识。
- **SQLite 数据存储**：
  - 使用 `sql.js` (SQLite WASM) 实现纯前端数据库存储。
  - 完美适配 **GitHub Pages** 静态部署。
- **排行榜系统**：
  - 自动记录登录用户的最高分。
  - 支持排行榜数据的 **JSON 导入与导出**，方便玩家之间进行成绩比拼。
- **高度可定制化**：
  - 游戏内支持自定义按键绑定。
  - 实时显示当前操作按键。

## 🛠️ 技术栈

- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **数据库**: SQLite (via `sql.js` WASM)
- **部署**: 兼容 GitHub Pages 静态托管

## 📂 文件结构

- `index.html`: 游戏入口/导航页。
- `snake.html`: 贪吃蛇游戏主体。
- `utils.js`: 核心工具类（包含 SQLite 数据库操作、用户认证、导入导出逻辑）。
- `home.js`: 首页 UI 交互逻辑。
- `script.js`: 贪吃蛇游戏核心引擎。
- `style.css`: 全局样式表。
- `vendor/`: 第三方依赖库（sql.js WASM 文件）。

## 📝 部署说明

1. 将所有文件上传至 GitHub 仓库。
2. 在仓库设置中开启 **GitHub Pages**。
3. 访问生成的链接即可开始游玩。
